# Lumon Industries - Employee Portfolio

A retro-futuristic portfolio website inspired by the interface design from Apple TV+'s *Severance*. This project serves as both a personal showcase and a tribute to the meticulous, corporate aesthetic of Lumon Industries.

## About

This project demonstrates my skills in web development, design systems, and creating immersive user experiences. The interface combines mid-century modernism with retro-futuristic technology to create a sleek, unsettling, yet functional portfolio.

## Features

- **Lumon OS Aesthetic**: Clean, minimalist design inspired by Severance
- **Interactive Navigation**: Tab-based content switching
- **Responsive Design**: Mobile-friendly layout
- **Performance**: Fast, static site generation with SvelteKit
- **Accessibility**: ARIA-compliant navigation and keyboard support

## Tech Stack

- **Framework**: SvelteKit 2.0
- **Styling**: Tailwind CSS with custom Lumon theme
- **Language**: TypeScript
- **Build Tool**: Vite
- **Deployment**: Static adapter for zero-server deployment

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
